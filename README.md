# 🍜 COCOiCO（ココイコ）

> もう「何でもいい」とは、言わせない。— 大学前のうまい店、見つけにいこ。

大学生向けの「今日どこ行く？を楽しくする」近所の飲食店レコメンド Web アプリです。
シチュエーションを選ぶだけで、大学生がガチで推す近所の名店に出会えます。

**HTML / CSS / JavaScript の単一ファイル**で動作し、外部依存ゼロ。`index.html` をブラウザで開くだけで使えます。

---

## ✨ 主な機能

| 機能 | 内容 |
| --- | --- |
| 🔥 **厳選モード** | AI「ココ助」とのチャットで、同行者・人数・時間帯・予算・気分・雰囲気・喫煙を対話形式で絞り込み、条件スコア上位3店を提示。デートや打ち上げなど「外せない」場面向け。 |
| 🎯 **探索モード** | 鉄板ターンテーブル（ルーレット）を回して、未知の名店をランダムに発見。全部 / 中堅 / 秘境 のスコープを選択。 |
| ✨ **店舗登録** | 行ってよかった店を登録。まだ誰も知らない店を登録すると、紙吹雪＋称賛モーダル＋「◯軒目の発見者！」＋開拓者カードでご褒美。 |
| 🧠 **町のご飯屋クイズ** | 4択クイズに正解して、解説から次に行く店へ誘導。連続正解で🔥コンボ、正解数で `グルメ見習い → 街グルメマスター → グルメ仙人` の称号が昇格。 |
| ☕ **隠れ家喫茶店** | 地図に載らない喫茶店を `読書向き / 完全禁煙 / 長居しやすい / 静か / 電源 / Wi-Fi` のタグで絞り込み。 |
| 🔗 **外部連携** | 各店から食べログ／ホットペッパーの検索ページへ遷移。 |

ポイント・登録店・クイズ成績・称号・お気に入りは `localStorage` に保存され、次回起動時も引き継がれます。

## 🎨 デザイン

コンセプトは **「Sizzle & Gold」** — 湯気の立つ屋台の高揚感を、金箔のように上質な金赤グラデで包んだ"美味しそうで止まらない"グルメ冒険アプリ。

- 暖簾ヘッダー（湯気アニメ）、スプラッシュ、とろみワイプのシーン遷移
- WebAudio による効果音（合成音・ミュート対応）、全要素が弾む「プリン・タップ」
- ルーレットの物理減衰＋磁石スナップ、店舗登録の紙吹雪＆開拓者カード
- ダークモード（夜の屋台）＆ `prefers-reduced-motion` 対応

## 🚀 使い方

1. このリポジトリをダウンロード（または clone）
2. `index.html` をブラウザでダブルクリックして開くだけ

サーバーやビルドは不要です。

## 🛠 技術

- HTML / CSS / JavaScript（バニラ、フレームワーク・外部CDN・画像なし）
- Canvas（ルーレット・紙吹雪）、WebAudio API（効果音）、localStorage（状態保存）

## 📝 補足（データについて）

掲載店舗は **食べログの掲載項目を模したサンプルデータ**です。実在店舗に架空の属性を付けないよう、店名はすべてサンプル名にしています。各店からは食べログ／ホットペッパーの検索ページへ遷移できます。
実データを使う場合は、別途取得した店舗データ（CSV/JSON）を `index.html` 内の `SEED` 配列に差し込む形で拡張できます。
